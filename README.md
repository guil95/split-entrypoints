# split-entrypoints

## How to use

### With bin file
```shell
 make run-local
```

#### Run http server
```shell
 ./split-entrypoints http
```

#### Run grpc server
```shell
 ./split-entrypoints grpc
```

#### Run consumers
```shell
 ./split-entrypoints consumers
```

### With docker

#### Run all the entrypoints
```shell
make run-docker
```
